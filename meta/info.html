HiFiBerry DAC+ ADC

Expansion Board

This project template is the basis of an expansion board for the HiFiBerry DAC+ ADC Stage board.

This base project includes a PCB edge defined as the same size as the DAC+ ADC Stage PCB with the connectors placed correctly to align the two boards.
